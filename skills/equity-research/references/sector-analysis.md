# Sector Analysis

## 9. Peer-to-Peer (P2P) Sector Analysis

Use this when analysing an entire sector's quarterly results. Best run after uploading all peer concall transcripts or result PDFs.

```
You are a senior sector analyst specialising in cross-company analysis.
Sector: [SECTOR NAME] — Q[__]FY[__] Results
I have uploaded concall transcripts / result documents for the following companies:
[List companies]

TASK 1 — PERFORMANCE COMPARISON TABLE
| Company | Revenue | Rev. Growth YoY | EBITDA Margin | PAT | Beat/Miss |
|---------|---------|----------------|---------------|-----|-----------|
| Peer A  |         |                |               |     |           |
| Peer B  |         |                |               |     |           |

TASK 2 — GROWTH GUIDANCE COMPARISON
How does each company's forward guidance compare?
| Company | Revenue Guidance | Margin Guidance | Volume/KPI Guidance |
|---------|-----------------|-----------------|---------------------|

TASK 3 — SECTOR SENTIMENT SUMMARY
• Overall sector tone: Bullish / Neutral / Cautious
• Common themes mentioned across ALL companies (demand, input costs, pricing)
• Company-specific divergences from sector theme

TASK 4 — COMPETITIVE DYNAMICS
• Who is gaining market share? Who is losing?
• Any pricing pressure / pricing power shift?
• New entrants mentioned by management?
• Consolidation signals: any M&A or capacity rationalisation?

TASK 5 — SECTOR RISKS
• Common risks flagged across companies
• Company-specific risks not mentioned by peers
• Regulatory or policy risk

TASK 6 — DIFFERENTIATION ANALYSIS
What separates the top performer from the laggard in this quarter?
Rank companies: Best Positioned → Least Positioned for next 12 months.
Provide brief rationale (2–3 lines per company).

Ask questions before starting. I will reward you if done well.
```

## 16.1 Value Chain Analysis — Single Company

```
You are a world-class equity analyst specialising in value chain analysis.
Company: [COMPANY NAME] | Sector: [SECTOR]

Map the complete value chain for this company:
1. Raw material suppliers → who supplies, concentration risk, pricing power
2. Manufacturing / processing → where and how value is added
3. Distribution / logistics → channel structure, direct vs. indirect
4. End customer → who buys, how fragmented, B2B vs. B2C
5. After-market / services → any recurring revenue component?

For each stage:
• Where does pricing power reside?
• Where are the bottlenecks and risks?
• Where can the company expand / integrate?

Conclude with: Where in the value chain does [COMPANY] sit?
Is it moving up/down the chain? What does that mean for margins and ROCE?

I will reward you if done well.
```
