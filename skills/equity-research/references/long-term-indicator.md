# Long-Term Indicator (LTI)

## LTI Framework — SOIC / Weinstein

The Long-Term Indicator (LTI) framework is used to assess whether a stock is technically positioned for a long-term entry. Based on the SOIC (Stocks On Investment Criteria) approach popularised by Basant Maheshwari and Stan Weinstein's Stage Analysis.

**Use LTI only AFTER fundamental conviction is established.** Technical confirmation is a secondary filter, not a replacement for fundamental research.

```
You are a long-term trend analyst using the VSTOP / CRS / Stage Analysis framework.
Stock: [TICKER] | NSE/BSE
Time horizon: 18–36 months

Analyse the stock using the following 7 criteria:

SECTION 1 — STAGE 2 BREAKOUT CONFIRMATION
• Is the stock currently in Stage 2 (Weinstein Stage Analysis)?
• Stage 2 = price above rising 200-DMA, trending higher with expanding volume
• Has it broken out from a base with volume confirmation?
• Duration of the base before breakout: __ weeks (longer = more powerful)
• If in a Stage 2 uptrend already: how far into Stage 2? (Early / Mid / Late)

SECTION 2 — RELATIVE STRENGTH vs. NIFTY 500 (CRS)
• Is the Comparative Relative Strength (CRS) line in new highs?
• Has CRS been trending up consistently over the last 6 months and 12 months?
• Is the stock outperforming or underperforming the Nifty 500 index?
• Rating: CRS Strong / Neutral / Weak

SECTION 3 — VSTOP (VOLATILITY STOP)
• Is the VSTOP indicator positive (price above volatility stop)?
• Has VSTOP flipped positive recently?
• VSTOP negative = do not enter / exit existing position

SECTION 4 — ADX (AVERAGE DIRECTIONAL INDEX)
• ADX value: __ (≥ 20 indicates a trending market)
• ADX < 20 = no clear trend; avoid new entries
• ADX > 25 = strong trend confirmed
• DI+ vs. DI− : positive alignment?

SECTION 5 — RSI (RELATIVE STRENGTH INDEX)
• RSI value: __ (14-period)
• RSI ≥ 45 required (trending stock should hold above 45)
• RSI ≥ 60 in strong uptrend
• RSI < 45 with declining price = weakness, avoid entry

SECTION 6 — ENTRY ZONES & RISK-REWARD
• Ideal entry: pullback to 50-DMA or support zone within Stage 2
• Entry price: ₹__
• Initial stop-loss: ₹__ (below which structure / 200-DMA)
• Trailing stop: most recent significant swing low
• 12-month price target (technical): ₹__
• Risk-Reward ratio: __ : 1 (must be ≥ 2:1 for valid entry)

SECTION 7 — LTI VERDICT
| Criterion | Status | Comment |
|-----------|--------|---------|
| Stage 2 confirmed | ✅/❌ | |
| CRS new highs | ✅/❌ | |
| VSTOP positive | ✅/❌ | |
| ADX ≥ 20 | ✅/❌ | |
| RSI ≥ 45 | ✅/❌ | |
| R:R ≥ 2:1 | ✅/❌ | |
| **LTI Overall** | GO / WAIT / AVOID | |

GO = All 6 criteria met — strong entry signal
WAIT = 4–5 criteria met — wait for remaining confirmation
AVOID = < 4 criteria met — do not enter

I will reward you if done well.
```

### LTI Monitoring Protocol (Once Position Entered)

Review quarterly:
1. **Exit signals**: Stage 3/4 transition OR 200-DMA turns flat OR CRS breaks down
2. **Trail stop**: Update to most recent significant swing low
3. **Review**: Has fundamental thesis changed? If not, hold.

### LTI Entry vs. Exit Summary

| Action | Trigger |
|--------|---------|
| Enter | All 6 LTI criteria met |
| Add to position | Pullback to 50-DMA in Stage 2, all criteria still valid |
| Partial exit | Stage 3 signals appear (distribution volume, CRS weakening) |
| Full exit | 200-DMA flat/declining, price below 200-DMA for 2 weeks |
| Hard stop | Price closes below initial stop-loss level |
