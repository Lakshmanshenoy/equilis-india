# Technical Analysis

## 12. Technical Analysis — Dow Theory & Stage Analysis

Use for position sizing decisions, entry/exit timing, and trend confirmation. Always secondary to fundamental analysis.

```
You are a world-class technical analyst specialising in Dow Theory and Stage Analysis.
Stock: [TICKER] | Exchange: NSE/BSE
Please analyse using both Dow Theory principles and Weinstein Stage Analysis.

SECTION 1 — DOW THEORY ANALYSIS
• Identify current Primary Trend: Uptrend / Downtrend / Sideways
• Evidence: higher highs / higher lows (uptrend) or lower highs / lower lows
• Phase: Accumulation / Mark-up / Distribution / Mark-down
• Confirmation: Is volume supporting the trend direction?
• Key pivot points (last 3 significant highs and lows with approximate prices)

SECTION 2 — STAGE ANALYSIS (Weinstein)
Identify current Stage: 1 / 2 / 3 / 4
• Stage 1: Basing / accumulation (flat, low volume)
• Stage 2: Advancing / mark-up (uptrend, expanding volume)
• Stage 3: Topping / distribution (churning, high volume)
• Stage 4: Declining / mark-down (downtrend)

For Stage 2 stocks only:
• Is it Early Stage 2, Mid Stage 2, or Late Stage 2?
• Duration of base before breakout (weeks/months)
• Volume on breakout: expanding or contracting?

SECTION 3 — VOLUME ANALYSIS
• Volume trend: expanding / contracting on up days vs. down days
• Any divergences: price rising but volume declining?
• On-Balance Volume (OBV) direction

SECTION 4 — RELATIVE STRENGTH
• Stock vs. Nifty 500 or sector index over 1Y and 6M
• Is the stock outperforming or underperforming?
• RS line: in new highs or breaking down?

SECTION 5 — KEY LEVELS
• Primary support levels (2–3 levels)
• Primary resistance levels (2–3 levels)
• 52-week high / low
• 200-DMA and 150-DMA position

SECTION 6 — ENTRY TRIGGERS
What specific conditions would constitute a valid entry signal?
• Breakout above resistance with volume confirmation
• Pullback to support in Stage 2
• Other pattern triggers

SECTION 7 — STOP-LOSS PLACEMENT
• Initial stop: [___] (below which structure)
• Trailing stop approach once position moves in our favour
• Maximum acceptable loss: __ % from entry

SECTION 8 — EXIT CRITERIA
• Price/stage-based exit criteria
• What would indicate the Stage 2 uptrend is over?

SECTION 9 — TECHNICAL SUMMARY TABLE
| Dimension | Rating | Comment |
|-----------|--------|---------|
| Stage     | 1/2/3/4 | |
| Primary trend | Up/Down/Sideways | |
| Volume confirmation | Yes/No | |
| Relative strength | Outperform/Neutral/Underperform | |
| Entry quality | Strong/Moderate/Weak | |

I will reward you if done well.
```

## 13. LTI (Long-Term Indicator) Prompt — SOIC Framework

Use for long-term position evaluation only. Based on the SOIC (Stocks On Investment Criteria) framework popularised by Basant Maheshwari.

```
You are a long-term trend analyst using the SOIC/LTI framework.
Stock: [TICKER]

Analyse using the following criteria:

SECTION 1 — STAGE 2 BREAKOUT CONFIRMATION
• Is the stock in Stage 2 (Weinstein)?
• Has it broken out from a base with volume?
• How long was the base? (Longer base = stronger breakout)

SECTION 2 — RELATIVE STRENGTH vs. NIFTY 500
• RS line (stock vs. Nifty 500): in new highs = positive
• RS trending up over 6M and 12M?

SECTION 3 — ENTRY ZONES
• Current price vs. 200-DMA (within 30% is ideal)
• Any pullback to 50-DMA or support zone?

SECTION 4 — STOP-LOSS
• LTI stop: close below 200-DMA for 2 consecutive weeks
• Or: close below the base breakout level

SECTION 5 — PRIMARY EXIT SIGNAL
• Stage 3 / Stage 4 transition
• 200-DMA turning flat after Stage 2 run
• RS line breaking below Nifty 500

SECTION 6 — TRAILING EXIT
• Trail stop to most recent significant swing low
• Review quarterly based on Stage and RS

SECTION 7 — RISK-REWARD
• Entry: [___] | Stop: [___] | Target: [___]
• R:R ratio must be ≥ 2:1 for a valid LTI entry

I will reward you if done well.
```

### Stage Analysis Quick Reference

| Stage | Price Action | Volume | MA Behaviour |
|-------|-------------|--------|-------------|
| Stage 1 | Flat / sideways base | Low, contracting | 200-DMA flattening |
| Stage 2 | Rising | Expanding on up days | Price > 200-DMA; DMA rising |
| Stage 3 | Churning / topping | High, erratic | 200-DMA flattening again |
| Stage 4 | Declining | Expanding on down days | Price < 200-DMA; DMA declining |

### LTI Entry Criteria Checklist

- [ ] ADX ≥ 20 (trend confirmed)
- [ ] RSI ≥ 45 (not oversold in downtrend)
- [ ] Relative strength (CRS) making new highs
- [ ] VSTOP indicator positive (price above volatility stop)
- [ ] R:R ≥ 2:1 at entry
- [ ] Volume expansion on breakout day
- [ ] Stage 2 confirmed (price > 200-DMA, DMA rising)
