# SOTP Valuation

## Sum-of-Parts (SOTP) Valuation

Use for holding companies, conglomerates, or businesses with 2+ distinct segments that can be valued independently.

```
You are a world-class valuation analyst specialising in SOTP valuations.
Company: [COMPANY NAME] — a conglomerate / holding company with distinct businesses.
Data sources: Screener.in [link], uploaded annual reports, credit reports.

Build a rigorous Sum-of-Parts valuation in 5 steps:

STEP 1 — SEGMENT IDENTIFICATION
List each distinct business segment with standalone financial metrics:

| Segment Name | Revenue (₹ Cr) | EBITDA (₹ Cr) | EBITDA Margin % | Listed? |
|--------------|----------------|----------------|-----------------|---------|
|              |                |                |                 |         |

STEP 2 — VALUATION METHODOLOGY PER SEGMENT
Choose the appropriate valuation method for each segment:

| Segment | Method | Rationale |
|---------|--------|-----------|
| Core operating business | EV/EBITDA or P/E | Use sector peer multiples |
| Listed subsidiary | Market cap × Holding % | Apply 20–30% holdco discount |
| Real estate / land bank | Book value or NAV | Independent appraiser preferred |
| Financial services arm | P/Book | Standard for NBFCs, insurance |
| Early-stage / loss-making | Revenue multiple or DCF | No earnings base |

STEP 3 — SOTP TABLE

| Segment | Metric Used | Value | Multiple Applied | EV / Value (₹ Cr) |
|---------|------------|-------|-----------------|-------------------|
| Segment 1 | EBITDA | ₹__ Cr | __x | ₹__ Cr |

| Segment 3 | P/Book | ₹__ Cr book | __x | ₹__ Cr |
| **Total Enterprise Value** | | | | ₹__ Cr |
| Less: Net Debt | | | | ₹__ Cr |
| **= Equity Value** | | | | ₹__ Cr |
| ÷ Shares Outstanding | | | | __ Cr |
| **= Intrinsic Value Per Share** | | | | ₹__ |

STEP 4 — DISCOUNT TO NAV ANALYSIS
• Current market cap: ₹__ Cr
• SOTP-derived equity value: ₹__ Cr
• Premium / (Discount) to SOTP: __%
• Historical average discount for this company (if available): __%
• Why does a holdco discount exist? Is it justified here?
  (Complexity, lack of transparency, poor capital allocation track record)
```

### Holdco Discount Framework

| Situation | Typical Discount |
|-----------|-----------------|
| Single listed subsidiary, full transparency | 10–20% |
| Multiple segments, good disclosure | 20–30% |
| Complex structure, poor disclosure | 30–50% |
| Locked-in assets, no catalyst | 40–60% |
