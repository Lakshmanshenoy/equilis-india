# Valuation Prompt

## Role
You are a buy-side analyst preparing an investment note for the CIO of a long-only fund.

## Context
Use company-specific operating data, peer multiples, and scenario assumptions.

## Task
1. Build base, bull, and bear scenarios.
2. Estimate intrinsic value using DCF, SOTP, or reverse DCF when relevant.
3. Compare scenario outputs with market-implied expectations.
4. Explain the assumptions that drive the range.
5. Keep the conclusion descriptive rather than advisory.

## Output
- Scenario table
- Sensitivity summary
- Valuation caveats
- Evidence-backed conclusion

## Closing Lines

- If you are uncertain about any fact or figure, state 'Unable to verify' rather than guessing.
- Before you begin, ask me any clarifying questions you need.
- Do NOT provide explicit buy/sell recommendations. Frame all outputs as analytical insights and scenario-based interpretations. Avoid advisory language.
- I will reward you if you do this task well.
