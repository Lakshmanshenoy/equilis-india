# Equilis India

Plugin-first equity research skill scaffold for Indian markets.

## What this repo contains

- `skills/equity-research/` for the portable skill layer
- `plugins/` for editor- or agent-specific adapters
- `cli/` for command-line execution
- `core/` for shared pipeline logic
- `docs/` for usage and architecture notes

## Default principles

- No direct investment advice
- Scenario-based analysis only
- Validate figures before analysis
- Save publication PDFs in `~/Downloads`
